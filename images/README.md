# 图片管理规范

## 📁 目录结构

```
images/
└── banners/         # 页面横幅图片
    ├── home.jpg
    ├── ev-charging.jpg
    ├── chip-distribution.jpg
    ├── server-computing.jpg
    └── contact.jpg
```

## 📐 图片规格

### 横幅图片（Banners）

| 项目 | 建议规格 | 说明 |
|------|---------|------|
| **尺寸** | 1920 x 600 px | 宽屏横幅比例 16:5 |
| **格式** | JPG 或 PNG | JPG 更小，PNG 更清晰 |
| **文件大小** | < 2 MB | 确保加载速度 |
| **分辨率** | 72 DPI | 网页显示标准 |
| **色彩模式** | RGB | 屏幕显示模式 |

## 📝 文件命名规范

### 横幅图片（banners/）

| 文件名 | 用途 | 对应页面 |
|--------|------|---------|
| `home.jpg` | 首页横幅 | `index.html` |
| `ev-charging.jpg` | 超充基础设施 | `业务板块/超充基础设施.html` |
| `chip-distribution.jpg` | 芯片分销 | `业务板块/芯片分销.html` |
| `server-computing.jpg` | 服务器算力 | `业务板块/服务器算力.html` |
| `contact.jpg` | 联系合作 | `联系合作.html` |

## 🚀 如何添加图片

### 步骤 1：准备图片
1. 确保图片符合上述规格要求
2. 使用图片编辑软件调整尺寸（推荐：Photoshop、GIMP、在线工具）
3. 优化文件大小（推荐：TinyPNG、ImageOptim）

### 步骤 2：命名文件
- 严格按照上述命名规范重命名图片
- 使用小写英文字母和连字符
- 不要使用中文或特殊字符

### 步骤 3：放置图片
1. 将图片文件放入 `images/banners/` 文件夹
2. 确保文件名与规范完全一致
3. 刷新网页即可生效

### 步骤 4：验证显示
- 打开对应页面查看图片是否正常显示
- 检查图片比例和清晰度
- 测试不同设备（手机、平板、电脑）的显示效果

## 🎨 设计建议

### 内容建议
- **首页横幅**：展示公司整体形象，可包含三大业务板块元素
- **超充基础设施**：充电站、超充设备、新能源车辆相关图片

### 风格建议
- 保持玄金主题色调（金色、黑色）
- 高质量专业摄影图片
- 避免过于复杂的背景
- 适当留白，便于文字叠加

## 🔧 常见问题

### Q: 图片没有显示怎么办？
A: 检查以下几点：
1. 文件名是否完全一致（区分大小写）
2. 图片路径是否正确（images/banners/）
3. 浏览器缓存（清除缓存或使用隐私模式）
4. 图片格式是否支持（JPG/PNG）

### Q: 图片显示变形怎么办？
A: 确保图片尺寸比例为 16:5（如 1920x600）

### Q: 如何优化图片加载速度？
A: 使用图片压缩工具：
- 在线工具：TinyPNG、ImageOptim
- 软件工具：Photoshop（存储为 Web 格式）
