## 🔧 技术实现细节

### ECharts配置
- **CDN版本：** ECharts 5.4.3
- **主题：** 玄金水墨主题（Ink-Gold Theme）
- **配置文件：** `/js/echarts-config.js` (695行)

### 统一管理机制

#### 初始化函数
所有图表使用 `initInkGoldChartResponsive()` 函数统一初始化：
```javascript
const chart = initInkGoldChartResponsive('chartId', {
    // ECharts配置
});
// 自动加入chartInstances数组，统一管理响应式
```

#### 响应式处理
```javascript
const resizeCharts = debounce(function() {
    chartInstances.forEach(chart => {
        if (chart) {
            try {
                chart.resize();
            } catch (e) {
                console.error('Chart resize error:', e);
            }
        }
    });
}, 200);

window.addEventListener('resize', resizeCharts);
window.addEventListener('orientationchange', resizeCharts);
```

## 📞 问题反馈

如发现任何图表显示问题，请检查：
1. 浏览器控制台是否有JavaScript错误
2. echarts-config.js是否正确加载
3. 图表容器div是否存在且ID正确
4. 网络连接是否正常（CDN加载）

**常见问题：**
- ❌ 图表不显示：检查容器ID是否匹配
- ❌ 样式错乱：检查玄金水墨主题.css是否加载
- ❌ 响应式失败：检查echarts-config.js是否正确引用
